1. What is DOM and what is the purpose of DOM?
ans: The Document Object Model (DOM) is a programming interface for web documents The DOM defines a standard for accessing documents: "The W3C Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."

2. How will you select HTML elements using DOM? Name the DOM methods.
ans:
1. getElementById(): This method selects an element with a specific ID attribute value
2. getElementsByClassName(): This method selects all elements with a specific class attribute value.
3. getElementsByTagName(): This method selects all elements with a specific tag name.
4. querySelector(): This method selects the first element that matches a specific CSS selector.
5. querySelectorAll(): This method selects all elements that match a specific CSS selector.

3. What is event bubble?
ans:Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

4. What is a callback function and why will you use it?
ans:A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
